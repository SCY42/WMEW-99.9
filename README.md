# 용도

이 프로그램은 비디오 게임 Mewgenics에 등장하는 가상의 라디오 'WMEW 99.9'(를 재구성한 무언가)를 시스템 콘솔에서 실행합니다. 음악뿐만 아니라 뉴스, 콜 레터, 음악 코멘터리, 그리고 광고까지 청취할 수 있습니다.

# 설치 방법

저작권 보호를 위해, 재생할 오디오 파일은 직접 마련해야 한다는 점에 유의하세요.

## 오디오 파일

> [!IMPORTANT]
> `Steam\steamapps\common\Mewgenics` 경로에 위치한 `resource.gpak` 파일에 포함된 에셋을 추출하세요.

- 스팀에서 Mewgenics 우클릭 → 관리 → 로컬 파일 탐색을 선택하여 빠르게 폴더를 찾을 수 있습니다.

- 추출 방법은 상관없지만, 저는 [GPAK-Extractor](https://github.com/ShootMe/GPAK-Extractor)를 사용했으며, 추출 결과물도 해당 툴을 기준으로 설명합니다.

- 추출된 `audio\music\radio` 폴더만 필요합니다. 다른 말로, `audio` 폴더 안의 `music` 폴더 안의 `radio` 폴더요.

## WMEW 99.9.exe

1. [릴리즈 페이지](https://github.com/SCY42/WMEW-99.9/releases)에서 Assets 탭 아래에 있는 `WMEW_99.9.zip` 압축 파일을 다운받으세요.

1. 압축을 해제한 뒤 이전 단계의 `radio` 폴더를 `WMEW_99.9` 폴더 안으로 옮기세요.

이후 `WMEW_99.9` 폴더는 원하는 위치로 이동시켜도 무방하지만, `WMEW_99.9` 안 파일들의 이름을 바꾸거나 옮기지 마세요.

# 팁

## 음량 조절

이 프로그램은 자체적으로 음량을 조절하는 기능이 없습니다. 시스템 볼륨 믹서에서, 개별 앱 음량 조절 기능을 통해 조절해주세요. 윈도우 기준, 음량 아이콘을 우클릭하면 '볼륨 믹서 열기' 옵션이 표시됩니다.

## 잘못된 음악 파일 이름

5개의 음악은, 그 음악에 맞는 인트로/아우트로 파일과 이름이 통일되지 않아 인트로/아우트로가 재생되지 않는 문제가 있습니다. 사소한 문제지만, 모든 인트로/아우트로를 제대로 감상하고 싶으시다면 `radio\songs` 폴더 안의 다음 파일들의 이름을 변경하세요.

1. `crystaline_dreams` → `crystalline_dreams`

1. `final_boss` → `dig_your_own_grave`

1. `fleas_and_parasites_v1` → `fleas_parasites`

1. `the_crack_of_doom` → `crack_of_doom`

1. `you_know_who` → `guillotina`
