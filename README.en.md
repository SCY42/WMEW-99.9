# Overview

This simple program simulates 'WMEW 99.9', a fictional radio station featured in the video game Mewgenics, through the system console. In addition to music, you can also listen to news reports, call letters, music commentary, and even commercials.

# Installation

Please note that you must set up the audio files yourself, due to copyright restrictions.

## Audio Files

> [!IMPORTANT]
> Extract the assets contained in the `resource.gpak` file, located in `Steam\steamapps\common\Mewgenics`.

* You can quickly locate the folder by right-clicking Mewgenics in Steam тЖТ Manage тЖТ Browse Local Files.

* Any extraction method is acceptable, but I used [GPAK-Extractor](https://github.com/ShootMe/GPAK-Extractor), and the following instructions assume its output structure.

* You only need the extracted `audio\music\radio` folder. In other words, the `radio` folder inside the `music` folder inside the `audio` folder.

## WMEW 99.9.exe

1. Download the `WMEW_99.9.zip` archive from the Assets section on the [Releases page](https://github.com/SCY42/WMEW-99.9/releases).

2. After extracting the archive, move the `radio` folder from the previous step into the `WMEW_99.9` folder.

After that, you may move the `WMEW_99.9` folder anywhere you like, but do not rename or relocate any files inside it.

# Tips

## Volume Control

This program does not include its own volume control feature. Please adjust the volume through your system's volume mixer using the per-application volume controls. On Windows, you can access this by right-clicking the volume icon and selecting тАЬOpen Volume Mixer.тАЭ

## Incorrect Music File Names

Five music tracks have inconsistent filenames that prevent their corresponding intro/outro files from playing correctly. This is only a minor issue, but if you want to enjoy every intro and outro properly, rename the following files inside the `radio\songs` folder:

1. `crystaline_dreams` тЖТ `crystalline_dreams`

2. `final_boss` тЖТ `dig_your_own_grave`

3. `fleas_and_parasites_v1` тЖТ `fleas_parasites`

4. `the_crack_of_doom` тЖТ `crack_of_doom`

5. `you_know_who` тЖТ `guillotina`
